# Como abrir a app da Proxiassistant

Esta primeira versão é uma app local simples. Não precisa de API, login ou instalação.

## Abrir no Mac

1. Abrir a pasta:

   `Proxiassistant - Redes Sociais/app`

2. Abrir o ficheiro:

   `index.html`

3. O ficheiro deve abrir no navegador.

## Como usar

1. Escrever um pedido no campo do Agente Coordenador.
2. Escolher o tipo de objetivo: Conteúdo, Campanha, Métricas, Concorrência ou Resposta a cliente.
3. Clicar em "Enviar ao Coordenador".
4. Ver os agentes acionados.
5. Rever os rascunhos gerados.
6. Aprovar, manter em revisão ou rejeitar.

## O que esta versão faz

- Simula o Agente Coordenador.
- Aciona agentes internos com base no pedido.
- Gera rascunhos de posts, legendas, CTAs e hashtags.
- Mantém tudo como "Em revisão" até aprovação humana.
- Guarda dados no navegador através de localStorage.
- Permite exportar os dados em JSON.

## O que ainda não faz

- Não liga ao Instagram.
- Não publica automaticamente.
- Não responde mensagens.
- Não usa API de inteligência artificial.
- Não gera imagens reais.
- Não guarda dados numa base de dados externa.

Essas partes podem ser adicionadas em fases futuras.
